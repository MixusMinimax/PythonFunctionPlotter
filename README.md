# Function Plotter

Plots functions.

## Usage:

type an expression and press enter

### Functions:
 - basic python operators
     - +
     - -
     - *
     - /
     - **
     - %
 - x              # the function's input
 - c              # real constant
 - sin(h(x))
 - cos(h(x))
 - exp(h(x))
 - ln(h(x))
 - derivative h(x)
 - random(h(x))   # h(x) is used as the seed
 - t              # current time in seconde (frame / 60)

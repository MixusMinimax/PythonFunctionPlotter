# Function Plotter

Plots functions.
Type an expression and press enter

## Features:

|    Function     |       Description        |
|-----------------|--------------------------|
| x               | the function's input     |
| c               | real constant            |
| sin(h(x))       |                          |
| cos(h(x))       |                          |
| exp(h(x))       |                          |
| ln(h(x))        |                          |
| derivative h(x) |                          |
| random(h(x))    | h(x) is used as the seed |
| t               | current time in seconde  |
